# Cardmarket History Dashboard 1.0.0

> **100 % AI-generated / Best Effort:** Alle Projektartefakte wurden mit KI-Systemen generiert.
> Trotz Tests kann die Software Fehler enthalten. Keine Gewaehr; Details:

Erste stabile Veroeffentlichung des lokalen Sales/Purchase-Dashboards fuer Cardmarket-
Statistikexporte.

## Highlights

- Import von Cardmarket `.xls`, `.xlsx` und semikolongetrennten `.csv`
- Verknuepfte Artikel, Produkte, Sendungen und Kauf-/Zahlungsereignisse
- Atomarer Neuaufbau und wiederholbarer Monatsimport
- Filterbares Dashboard, Monatsdiagramm, Explorer und CSV-Zeitraumreport
- Datenqualitaetspruefungen und nachvollziehbare Rohquellen
- Lokaler read-only Betrieb mit maskierten Personendaten
- Sichtbarer 100-%-AI-generated- und Best-Effort-Hinweis
- MIT-Lizenz

## Wichtiger Upgrade-Hinweis

Version 1.0.0 verwendet Normalisierungsversion 3. Datenbanken aus Vorabversionen muessen nach
dem Upgrade aus den unveraenderten Originalexporten neu aufgebaut werden:

```powershell
.\.venv\Scripts\python -m cm_dashboard.cli rebuild `
  --source ".\exports" --db ".\data\cardmarket.db"
.\.venv\Scripts\python -m cm_dashboard.cli validate `
  --db ".\data\cardmarket.db"
```

Vorher den Webserver stoppen und Datenbank, Originalexporte sowie `accepted_issues.json`
sichern.

## Grenzen

Der Report ist kein Steuer-, Gewinn- oder Inventarreport. Mehrbenutzerbetrieb, Hosting,
Web-Uploads, automatische Backups, Waehrungsumrechnung und FIFO sind nicht Bestandteil von
Version 1.0.0.
